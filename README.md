# VAST Challenge 2019: Mini-Challenge 1

Решение первой мини-задачи конкурса VAST Challenge 2019, посвященной анализу сообщений граждан о повреждениях после землетрясения в городе Сент-Химмарк.

## Обзор задачи
Цель проекта — использовать методы визуальной аналитики для приоритизации районов реагирования экстренных служб. Основная сложность заключается в сопоставлении официальной карты толчков (ShakeMap) с субъективными и зашумленными отчетами жителей.

## Ключевые вопросы и инсайты

### 1. Расстановка приоритетов и наиболее пострадавшие районы
Первоначальная карта толчков не всегда совпадает с реальными разрушениями. Анализ позволил выявить районы с критическими повреждениями инфраструктуры, которые требовали немедленного вмешательства.

### 2. Анализ неопределенности и надежность отчетов
Данные от граждан (crowdsourcing) содержат много шума. В работе проведено сравнение надежности отчетов по разным районам для фильтрации ложных сигналов.

### 3. Динамика изменений во времени
* **Развитие ситуации:** В первые часы данные поступали хаотично, со временем очаги повреждений локализовались.
* **Изменение неопределенности:** С накоплением отчетов неопределенность в «активных» районах снижалась.
