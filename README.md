# VAST Challenge 2019: Mini-Challenge 1

Решение первой мини-задачи конкурса VAST Challenge 2019, посвященной анализу сообщений граждан о повреждениях после землетрясения в городе Сент-Химмарк.

## Обзор задачи
Цель проекта — Анализ последствий землетрясения в городе Сент-Химмарк.

## Ключевые задачи

### Задача 1. Приоритизация районов для реагирования
Службы экстренного реагирования будут основывать свой первоначальный ответ на карте толчков землетрясения. Используйте визуальную аналитику, чтобы определить, как их ответ должен измениться на основе сообщений о повреждениях от граждан на местах. Как бы вы расставили приоритеты в отношении районов для реагирования? Какие части города пострадали больше всего? 

### Задача 2. Оценка надежности и неопределенности данных
Используйте визуальную аналитику, чтобы показать неопределенность в данных. Сравните надежность отчетов районов. Какие районы предоставляют надежные отчеты? Предоставьте обоснование своего ответа.

### Задача 3. Анализ временной динамики
Как условия меняются со временем? Как неопределенность меняется со временем? Опишите ключевые изменения, которые вы видите.
